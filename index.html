ProgressHR

ProgessHR is a small javascript widget which creates 'progress aware' <hr> line breaks in text. It provides visual feedback which gives users a sense of the length of an article, saving them from having to scroll the page to peek. (Because who knows whether the article runs all the way to the bottom of a page?)

Usage

...

Code

function init() {
    ProgressHR();
}
window.onload = init;
...
